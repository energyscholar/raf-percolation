# Critical Thresholds in Autocatalytic Network Formation: A Percolation Perspective

Companion repository for the paper investigating the relationship between RAF (reflexively autocatalytic, food-generated) set emergence thresholds and percolation theory.

## Structure

- `paper/` — LaTeX source
- `scripts/` — Numerical analysis and simulations
- `figures/` — Generated figures
- `notes/` — Working notes (not included in paper)

## Building the paper

```bash
cd paper
make
```

## Dependencies

- Python 3.10+
- NumPy, SciPy, Matplotlib
- LaTeX (texlive-full or equivalent)
